Breaking: Everything Normal

This article does not have its own variable scope, which means that changing the global style.color will change its color. It's just your average div that don't want no trouble.

Shocking REDelations

This color variable is normally set to blue, but because this is inside an article tag it is given a different scope - a red scope.

Change We Can Believe In

This article has an ID of 'changeMe', which makes it easier for us to grab it in the code and change its scoped value
